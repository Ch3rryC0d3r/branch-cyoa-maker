What is a Leaf?

A Leaf is a single option line inside a Node. Think of it as a branch of choices the player can pick. Each Leaf can show text, point to another Node, check conditions, and perform actions.

Leaf Structure

A Leaf is written in this format:

Text | Next | Condition | Actions
  • Text ‚Üí What the player sees as the choice.
  • Next ‚Üí Which Node to go to after choosing (number or variable).
  • Condition ‚Üí Optional check before showing this option.
  • Actions ‚Üí Things that happen if chosen (like changing variables or items).

Example:

Take the sword | 5 | has_item:Key | add_item:Sword

This shows the choice ‚ÄúTake the sword‚Äù. It goes to Node 5, only appears if the player has a Key, and adds a Sword to their inventory.

Conditions

Conditions decide whether the player sees the option. If the condition is not met, the Leaf is hidden.

  • has_item:Sword ‚Üí Only show if the player has Sword.
  • not_has_item:Key ‚Üí Only show if the player doesn‚Äôt have Key.
  • CLICKS==1 ‚Üí Show if variable CLICKS equals 1.
Open the gate | 10 | has_item:Key & var:CLICKS==1 | remove_item:Key

This option is only visible if the player has a Key and CLICKS is 1. It takes the Key and moves to Node 10.

Actions

When the player picks a Leaf, actions run instantly. They can:

  • set:Name=Value ‚Üí Sets a variable (e.g. set:CLICKS=1).
  • CLICKS+=1 ‚Üí Adds to a variable.
  • add_item:Sword ‚Üí Gives an item.
  • remove_item:Key ‚Üí Takes away an item.
  • rand_set:Weather:Rain,Sun ‚Üí Randomly sets Weather to Rain or Sun.
  • goto:5 ‚Üí Jump straight to Node 5.
Attack the guard | 12 | | CLICKS+=1 & add_item:Sword

Choosing this adds 1 to CLICKS, gives the Sword, and jumps to Node 12.

Tips

  • You can chain multiple actions with ; or &.
  • If there‚Äôs no Condition, the option always shows.
  • If there‚Äôs no Next, the story stays in the same Node.

Summary

Leaves are the ‚Äúchoices‚Äù in your Branch story. Each one has Text, a Next step, optional Conditions, and Actions. By combining them, you can create deep, interactive paths for your players.
